# Wrapper

inuitcss plugin that wrap content

## Installation

- npm : `npm install --save totem-wrapper`
- bower : `bower install --save totem-wrapper`

## Available classes

- `o-wrapper`

## Configurable variables

- `$wrapper-width`
- `$wrapper-guuter`

## Usage 

Basic exemple

```html
<div class="o-wrapper">
	<!-- child content -->
</div>
```
